## 📖 Acerca del Proyecto

**La Cuentoneta** es una plataforma abierta, colaborativa y sin fines de lucro creada para fomentar la lectura y difusión de literatura breve en formato digital.

El proyecto nació en la ciudad de **Santa Fe, Argentina**, durante los últimos días de diciembre de 2021, impulsado por un grupo de personas apasionadas por la literatura y la tecnología.

La iniciativa busca construir un espacio comunitario donde lectores y escritores puedan compartir historias cortas organizadas mediante **Storylists**, inspiradas en las playlists utilizadas por plataformas como Spotify o YouTube.

---

## 🎯 Misión

Crear una plataforma accesible, amigable y moderna que incentive la lectura digital y la difusión de relatos breves mediante experiencias innovadoras y colaborativas.

---

## 👁️ Visión

Convertirse en una referencia digital para la literatura breve en habla hispana, facilitando el acceso a contenidos literarios de calidad y fortaleciendo comunidades lectoras alrededor del mundo.

## 🚀 Instalación

### Clonar repositorio

```bash
git clone https://github.com/isair/la-cuentoneta.git
```

### Acceder al proyecto

```bash
cd la-cuentoneta
```

### Instalar dependencias

```bash
npm install
```

### Ejecutar entorno de desarrollo

```bash
npm run dev
```
